About

This is a spam mail detection webpage designed to check whether a given email is spam or not.

Programming Languages Used

  • HTML & CSS
  • Python Flask for Backend
  • Python for ML Model

Algorithm Used

Naive Bayes Classifier

About Dataset

The dataset consists of 1226 genuine mails and 1101 spam mails, totaling 2327 emails.

It was downloaded from Kaggle. You can check out the dataset here.
